#### 1. **核心绘图模块** (core/visualization/)

  - 可调整曲线间距

- ✅ `plot_shg_contour.m` - 二维等高线热图
  - 完整的热图绘制
  - 可选等高线显示
  - 灵活的颜色范围设置
  - 参数轴标签定制

#### 2. **数据导入模块** (core/data/)
- ✅ `import_shg_data.m` - 单文件导入
  - 自动波长轴生成
  - 灵活的字段识别

- ✅ `load_shg_power_data.m` - 功率标定加载
  - 支持双文件加载
  - 自动数据对齐

#### 3. **分析模块** (core/analysis/)
- ✅ `normalize_shg_spectrum.m` - 多种归一化方法
  - max 归一化
  - peak_range 归一化
  - area 归一化
  - none (无操作)

- ✅ `shg_power_fitting.m` - 功率依赖性拟合 **[重点功能]**

  - 指数模型: I = a × exp(b×P)

## 💡 核心功能详解

### plot_single_shg() 
**用途**: 绘制单条SHG光谱

```matlab
[fig, ax] = plot_single_shg(spectrum, wavelength, ...
    'Title', 'My SHG', ...
    'LineColor', 'r', ...
    'LogScale', false);
```

**特性**:
- 简洁的接口
- 支持对数Y轴
- 可调整所有视觉元素

---

### plot_shg_stack() 
**用途**: 多条光谱堆叠可视化（最常用功能）

```matlab
plot_shg_stack(SHGData, ...
    'WavelengthRange', [600 700], ...
    'NormalizeMethod', 'peak_range', ...
    'Spacing', 0.15, ...
    'Colormap', 'power');
```

**特性**:
- 5种归一化方法
- 灵活的波长范围选择
- 5种色彩映射
- 可调整曲线间距

---

### plot_shg_contour() 
**用途**: 二维参数空间热图

```matlab
plot_shg_contour(SHGData, ...
    'ContourLevels', 50, ...
    'Colormap', 'hot', ...
    'ColorRange', [0.5 1.0]);
```

**特性**:
- 直观的数据分布
- 可选等高线显示
- 灵活的颜色范围设置

---

### shg_power_fitting() 
**用途**: 功率依赖性分析 **[重点功能]**

```matlab
[fitresult, gof] = shg_power_fitting(power, intensity, ...
    'ModelType', 'power', ...
    'Plot', true);

% 结果:
% fitresult.model = 'I = 0.5 × P^2.01'
% gof.rsquare = 0.9987
```

**特性**:
- 幂律拟合 (I = a×P^n)
- 指数拟合 (可选)
- 自动拟合质量评估
- 可视化拟合结果

**物理意义**:
- 对于真正的SHG: n ≈ 2.0
- R² > 0.99: 优秀拟合
- 偏差原因: 非线性吸收、缺陷等

## 🔧 技术细节

### 数据结构标准
```matlab
SHGData.data           % (N×M) 矩阵
SHGData.axisscale{1,1} % (N×1) 功率向量
SHGData.axisscale{2,1} % (M×1) 波长向量
```

### 函数命名约定
- `plot_*` - 绘图函数
- `import_*` - 数据导入
- `load_*` - 数据加载
- `*_fitting` - 拟合分析
- `normalize_*` - 数据处理

### 参数传递
- 所有函数使用 inputParser
- 支持 Name-Value 参数对
- 严格的参数验证
